Проект для обработки СSV файлов с данными о производительности сотрудников и вывода отчёта.
1. services.py:
    - read_csv_file(files: list) - функция принмиает на вход список СSV файлов, обрабатывает их и возвращает список словарей с данными по средней производительности на каждой структурной позиции.
    - write_csv_file(report_data: list[dict], report_file: str) - функция принимает на вход обработанные данные и наименование файла, куда результаты будут сохранены, записывает данные в СSV файл и выводит их в консоль в формате таблицы.

Для запуска:
1. Установить зависимости из toml файла
2. В командной строке запустить main.py с обязательным указанием флагов "--files", "--report". Пример - "python main.py --files employees1.csv employees2.csv --report performace.csv"
